# Metadata Types

There are 5 basic Metadata types that can not be deleted:

* Device Name
* Device Owner
* Location
* Device Timezone
* Hotspot Name

Additional Metadata types available:

* Text
* Number
* Unit
* Time Range
* Contact
* Time
* Cost
* Coordinates
* Switch
* Device Reference
* List
* Table
* IMEI
* ICCID
